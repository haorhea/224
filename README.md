                        <div class="txt js-content ptag"><p>还要想标题啊，好烦啊，不写标题了，反正就是转生领养PA的承花，接上篇。</p> 
<p>&nbsp;</p> 
<p>&nbsp;</p> 
<p>“承太郎今年过生日想要什么礼物？”</p> 
<p>每年新年刚过的时候，赫莉都会问出这个问题来，儿子的生日在正月的末尾，提前一个月开始准备生日礼物总是不错的。小时候承太郎据实以答，想要模型飞机，想要遥控赛车，想要外公上次说过的威尼斯玻璃陀螺，过了三个星期之后多半能如愿以偿。年纪稍微上去一些，渐渐地不愿意把心里真正想要的东西说出来了，怕显得自己幼稚，于是压压帽檐丢一句“随便”，母亲便按着她的品味和意思给他买生日礼物，名牌衣履皮带，豪雅手表，铂金嵌钻的袖扣，诸如此类的奢侈品多半是这几年聚集起来的，有的合他的意，有的让他哭笑不得。合他意的，生日第二天就当着母亲的面穿戴起来，赫莉乔斯达是个性子很容易满足的温柔女性，儿子第二天用了她的礼物，尽管一脸的不以为意，但这也就完全能抵上一句热情的“谢谢”了。</p> 
<p>十八岁的生日是在埃及过的，没有特别庆贺，过得很匆忙，当时花京院给他弄来了蜡烛，白色的，不太吉利，往硬邦邦的粗粝点心上一插，颇有些羞涩地催他许愿。他笑笑，摘下帽子许了个愿望之后，吹了蜡烛，和花京院一人一半，把那块口感很差的点心给吃了，满嘴食物残渣的时候交换了一个甜得发腻的吻，心里的幸福快要满出来。</p> 
<p>十八岁的那个生日愿望完全没被实现，不如说是与现实背道而驰。</p> 
<p>十九岁的时候，母亲照例问他，他想了想说，不用了吧，不用了。赫莉后来还是给他买了新手表，理由是原来那块豪雅一年前就已经不走了，不知为什么承太郎一直不换掉。</p> 
<p>之后的几年里，他反复告诉母亲，不用了，不用给他买东西，真的不用。空条承太郎在十八岁后突然对自己有了新的认识，那趟旅程之前原本是天大地大没有我大的想法，什么好东西送到面前来都觉得理所当然，回程中却发现自己根本渺小得不行，甚至可以说是配不上别人的礼物，簇拥，信任，还有爱。</p> 
<p>二十三岁的新年，赫莉又抛出了同一个问题，母子两人正一起准备晚饭，竟然学会了做家务的空条承太郎正削着土豆皮，他看得出母亲只是形式上地随便一问，但是今年他却不打算说一句“不用”应付过去。</p> 
<p>“我……”他说出一个音节，讲到这里，却不知道后续该怎么继续说。赫莉听他难得提了要求，立刻很热情地问他要什么，语气里掩饰不住的高兴。</p> 
<p>“我想领养一个孩子。”</p> 
<p>他最终盯着土豆，说了出来，母亲愣了一下，也是，就算是开明如她，也不可能在听到二十三岁的儿子提出这样的要求之后立刻笑着回答“好啊那就去领一个”吧。母亲问他，是个怎样的孩子？承太郎在心里描摹着那些他每天晚上都拿出来反反复复看一遍的照片，组织着语言：红头发，五岁，日本人，很瘦小，有点太瘦了，亲生父母抛下了他……</p> 
<p>然而想到这里，又觉得这样的描述无论对他还是对赫莉都没什么意义，搪塞和逃避而已，母亲问“是个怎样的孩子”，其实当然也隐含另一个问题：为什么？</p> 
<p>他叹了口气，把削好的土豆扔进篮子，最后明白无误地说出了理由：</p> 
<p>“那孩子……是花京院。”</p> 
<p>“典明吗？”</p> 
<p>“嗯。”</p> 
<p>“他现在怎么样？”</p> 
<p>“五岁了，是孤儿，父母不知道在哪里，也不知道在不在世。”</p> 
<p>赫莉不说话了，承太郎也没有开口。他知道母亲很喜欢那个才来过家里一次的人，哪怕带回来的时候血淋淋的，她也一直擅自认定那次承太郎是带朋友回来了，只是花京院没有机会再次迈入空条家的门槛，少年一去就没有再回来。</p> 
<p>如果她追问起细节，承太郎是打算一口咬定“不管怎么样，我知道他就就是花京院”的，但是她最后笑了笑，把一篮子洗剥好的土豆送进锅里，很平静地说：“那就去领吧，别忘了带回来让我看看。”</p> 
<p>“……谢谢。”</p> 
<p>他也只能说谢谢。</p> 
<p>“承太郎，今年的生日礼物，得要你自己去提了。”</p> 
<p>“嗯。”</p> 
<p>.</p> 
<p>承太郎总是在某些时刻发现自己太天真，就好像当年觉得自己一定会带着同伴和恋人毫发无损地从埃及得胜归来，亦或是真的得胜归来之后，他总以为花京院就算死了，那个灵魂也总该在他身边的某一处默默地看着他，他甚至在最消沉的时候对着空气说一堆啰里啰嗦的话，以为花京院的幽灵会在身边听，用看不见摸不着的拥抱安慰一下被独自留在现世的自己。</p> 
<p>真是傻，其实那个灵魂早就去了别的地方，开始新的生活了。</p> 
<p>乔斯达家的信用和钱实在好用，否则他根本难以想象独身的二十三岁男人怎么可能顺顺利利地领养到一个五岁孩子。他在生日的前一天去了孤儿院，看到了花京院本人，一模一样的名字，熟悉的发色和瞳色，不太熟悉的五官长相，他试图最后一次确认眼前的人究竟是不是他要找的，放出白金之后，很欣慰地看到孩子身后蔓延出细小的绿色触手，花京院本人大概还没有察觉，白金之星看了他一眼，他对自己的替身点点头，接着对孤儿院的院长确认道：</p> 
<p>“是他。”</p> 
<p>这句话好像把几年来一直挂在心口的一块千斤巨石解了下来，就好像在一个徘徊已久的死胡同里猛然看见一截不知何时垂挂下来的绳梯。</p> 
<p>他对花京院说一周内来接他，心里想着他毕竟只有五岁，总该给他点时间接受适应这个消息，可回到家却坐立不安起来，收拾独居的房子收拾了一夜没有睡，第二天清晨照照镜子，看见的却是一张精神奕奕不见疲态的脸。</p> 
<p>我大概是太亢奋了，他心想，今天再去看看他也成。</p> 
<p>一去就索性就把他领了回来。承太郎小心翼翼地和养子保持着距离，几次伸出手，不敢碰到他，不敢揉他的头，不敢抱，心里总想着这些事当着所有人的面做，以前花京院都是很讨厌的，恋人自尊心太强，在公众场合想少许亲热一下就总会惹他生气。然而走了一路，五岁的孩子却主动伸手要抱，承太郎松了口气，一把提起和行李箱差不多重量的花京院托在臂弯里，对方一点也不抗拒，小小软软的手伸过来搂住的脖子，絮絮叨叨地开始跟他说话。</p> 
<p>花京院一遍遍叫他的名字，承太郎笑起来，一遍遍地答应，对话的时候，猛然发现自己之前一直都漏过了一个盲点。怀里这个并不是他在十七岁时死去的恋人花京院典明，说是转世也好，同一个灵魂也罢，就算心里执拗地反复强调“他就是花京院”，其实也不是了，他现在所抱着，所领养来的，只是个货真价实的五岁孩子。</p> 
<p>他们并不是，也不可能是同一个人。</p> 
<p>.</p> 
<p>到家已经过了晚餐时间。承太郎原本颇有一番雄心壮志，他这些年里已经学过几个拿手菜，花京院到了家里以后，当然要精益求精，让他每段都吃得营养又美味。然而这雄心壮志在第一天就受了挫折，回家的车程太长，还在路上的时候就到了饭点，花京院抱着他车上的海豚娃娃一句话没说，承太郎问他，饿不饿？他看了养父一眼，很小声地立刻答道，饿了。</p> 
<p>结果领来了之后，第一顿吃的是西式快餐，承太郎看着点餐面板上一溜的煎炸食品，对没算准时间的自己有点恼怒，他平时也不怎么吃这些，最后让花京院自己挑。孩子看了半天，最后选了里面夹鱼肉的汉堡，还问点餐的年轻女性，有没有橙汁？</p> 
<p>空条承太郎的脸是一张有用的通行证，就算他们磨叽了很久，快餐店的点餐员工也丝毫没有显露出一点不耐烦，她看着被承太郎抱在胸前的花京院，笑咪咪地说可以把套餐里的可乐换成橙汁，他们于是点了一样的两个套餐，承太郎的加大，可乐都换成果汁。</p> 
<p>他一手托着花京院，一手托着满满当当的托盘，找了个没什么人的地方落座，周围有目光和窃窃私语，他也没怎么注意。</p> 
<p>孩子选了鱼，鱼营养好，还换了橙汁，橙汁也比可乐健康，承太郎对花京院的选择已经有了点傻爸爸式的洋洋得意。他们一起吃了晚餐，吃完后花京院如约掏出蜡笔，在餐盘垫纸的背面给他画了张颇为孩子气的生日贺卡，他珍而重之地说谢谢，又折好了放进钱包。附近走动的服务人员早就不时瞟着他们这一桌，听到一大一小两个人说着什么生日快乐，凑过来推荐餐厅的生日活动，说可以免费提供蜡烛和许愿单，得知过生日的是大的那个之后有点讪讪地退后，不再说话，饶有兴致地打量着他们俩。</p> 
<p>换做高中时的承太郎，对这样的打量多半要爆发，现在却没心思多说什么了，托腮看着养子，看他把蜡笔一支一支收拾起来，本该是高兴的，心里却莫名地觉得有一丝难受，大概是因为刚才那个店员提起了他不愿意想到的事吧，和花京院共度的生日，许愿，吹蜡烛，像是缓缓沉入热水浴缸一样的幸福感，这样的经历过往只有一次，最后结果也不好，他不想再来一次了，想起来就有点心悸。</p> 
<p>晚饭后又开了三刻钟的车，终于是到了家里，进屋他开了灯，开了地暖，开了空调，搞得空荡荡的家里一片暖洋洋的才把花京院从怀里放到地上，一间间带他看过去，告诉他哪间是什么。孩子赤着脚跑来跑去，看完之后说了一句：</p> 
<p>“这么大啊！”</p> 
<p>的确是大，一般承太郎这个年纪的独居者也不会住这样大的房子，但他以前习惯了大房子，来之前给花京院安排了一间房间，还单独辟出了一间放童书和玩具的游戏室，大小都和孤儿院的游戏室差不多，原本觉得没什么不对劲，花京院这么一说，才后知后觉发现自己有点热情过头，好心办了错事。孩子站在地上，一只手捏着他的裤管，看上去震惊多过兴奋，可怜兮兮的样子，但是也没有提出异议。</p> 
<p>两个人住这么空荡荡的一间大房子，他应该是觉得不安了吧。</p> 
<p>承太郎不是没有想过让花京院和自己住一间，但是想想就过，最后很坚决地单独空出来一个新房间。虽说只是个五岁孩子，但毕竟是花京院，同床共枕这个习惯还是从一开始就不要养成比较好，他对自控力没那么自信，不想他们的关系早早地就开始变质。</p> 
<p>花京院最后接受了他的单人房间，一样一样地从行李箱里把东西拿出来搁到桌上，床上，架子上，中途就打起了哈欠。承太郎于是带他去洗澡，拆开全新的小尺寸毛巾睡衣洗漱用具，一件件给他换上用上，看着他爬上床，漫长的一天眼看着将要过去，承太郎说了晚安，花京院却不出声，虚着眼睛看着他。</p> 
<p>是有什么意见又不敢说吗，他从房间门口走回孩子的床边，轻轻地坐下了，顺一顺那红色的铺散在枕头上的头发，花京院皱起小脸，往被子里缩了缩。</p> 
<p>“怎么了？”</p> 
<p>没有回答，他也不气馁，心想和小孩子交流原本就是这样，于是低下头继续柔声问：</p> 
<p>“你晚上睡觉，要关灯还是开着灯？”</p> 
<p>“关灯。”</p> 
<p>“被子冷不冷？”</p> 
<p>“不冷。”</p> 
<p>“枕头会不会太高？”</p> 
<p>“不高，正好。”</p> 
<p>他这样问了好几项，得到的答案却都是满意，也不像是骗人，似乎没出什么差错，可花京院就是不闭上眼睛，承太郎想了一会儿之后问道：</p> 
<p>“要不要我给你讲故事？”</p> 
<p>原本半虚着的眼睛立马亮起来，花京院躲在被子里说，好。</p> 
<p>这建议有些自掘坟墓的意思，承太郎并不擅长说睡前故事，脑子里当然也储备着几个经典童话，睡美人啦，桃太郎啦，白雪公主啦，诸如此类的，但是都记得不太清楚了，现在多少有点犯难，他问花京院想听什么，花京院把脸从被子里露出来一点说，我带来了一本书，老师给我读到一半，你给我念这个吧。</p> 
<p>承太郎如获大赦，找到那本颇为破旧的书，根据花京院的要求，斜躺在他床边，一手撑着侧脸，一手翻书，就着昏黄的床头灯，给他说了小美人鱼的故事，孩子安安静静地躺在被窝里听，等说到了半途，他问承太郎：</p> 
<p>“我可以问问题吗？”</p> 
<p>“当然可以。”</p> 
<p>“你真好，”花京院感叹了一句，“在那里的时候，晚上听故事是不能问问题的。”</p> 
<p>“老师不让？”</p> 
<p>“不是，我们都是一房间的人一起听故事，我问了问题，其他小朋友会不耐烦。”</p> 
<p>“这里只有你，没关系，想问什么？”</p> 
<p>“王子为什么不记得小美人鱼的脸了？”花京院眨着眼睛，盯着承太郎，“她救了他的命啊，而且还爱上了她，为什么不记得她长什么样子了？”</p> 
<p>承太郎哑然，他思索了一阵，勉勉强强地说，多半是海难之后神志不清，所以记错了吧。</p> 
<p>“可是小美人鱼再次出现在他面前的时候，他也没认出来。”</p> 
<p>“是个糊涂的王子吧。”</p> 
<p>“真是糊涂。”</p> 
<p>“是啊。”</p> 
<p>花京院不再说话了，承太郎得以蒙混过关，继续把故事说了下去，越接近结局，越觉得不对味，读到了最后，果然和记忆力一样，是个悲剧。他合上书想看看花京院的反应，身边传来的却是均匀而微弱的呼吸声，花京院闭着眼睛，什么也没说。</p> 
<p>睡着了啊，承太郎有点庆幸，关了台灯准备离开，床上却传来口齿不清又轻又细的声音：</p> 
<p>“……承太郎。”</p> 
<p>“怎么了？”</p> 
<p>“你今天跟我一起睡好不好？我第一次一个人睡。”</p> 
<p>“……”</p> 
<p>“我有点害怕。”</p> 
<p>“……好吧。”</p> 
<p>他走过去，钻进花京院的被窝，这床原本就是成年人的尺寸，他躺进去也不嫌拥挤，两个人平躺在一起，互相都没有碰到对方。</p> 
<p>“晚安承太郎。”</p> 
<p>“嗯，晚安。”</p>

<p>太郎这时候是二十三，暂时还是黑色款，不敢对小孩动手动脚，一方面是因为心里紧张，一方面是因为老记得以前的花不喜欢他大庭广众动手动脚，所以不敢，要不要搞光源氏，他后面大概会纠结一下，后面慢慢长大的部分会写，不知道什么时候写，反正会写。</p> 
<p>花这时候是五岁，法皇已经有了，但是他自己没意识到，也没注意，造型和本来那个是一样的，就是小点，性格暂时就是普通五岁小孩性格，稍微比平均更聪明，更冷静一点，不过总之就是五岁小孩</p> 
<p>大概就这样，太郎怎么找到花的，不想写了，我们有SPW嘛，反正就找到了呀。</p>
